Tea Garden Guardian

Tea Garden Guardian is a wave-based top-down shooter originally developed in Processing (Java) for IAT 167 and later adapted to p5.js for web deployment.

The game places the player in the role of a tea spirit tasked with protecting a peaceful tea garden from progressively challenging enemy waves, culminating in a boss encounter.

This project explores object-oriented programming, game state management, enemy behavior design, and player feedback through visual effects, HUD elements, and power-up systems.

Controls

  • Move: W / A / S / D
  • Shoot: SPACE
