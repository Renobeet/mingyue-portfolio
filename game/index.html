Tea Garden Guardian

Tea Garden Guardian is a wave-based shooting game developed as part of IAT 167. The project explores enemy AI behavior, power-up systems, and state-based game design.

The game was originally built in Processing (Java) and later ported to p5.js for web deployment.

Controls

  • Move: W / A / S / D
  • Shoot: SPACE
